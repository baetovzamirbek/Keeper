# Keeper                                                                                                    
                                                                                                             
 ![Keeper screenshot](assets/keeper.png "Keeper Page")
                                                                                                             
 My clone of [Google Keep](https://keep.google.com) in NodeJS and ReactJS.
                                                                                                   
 ### Features

 - Note creating, deleting, updating
 - Note sharing
 - Note archiving
 - Activity log
                                                                                                             
 ### Requirements
                                                                                                             
 1. NodeJS
 2. MongoDB
